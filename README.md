## Run Locally

Clone the project

```bash
  git clone https://github.com/zaidajani/startupeer.git
```

Go to the project directory

```bash
  cd startupeer
```

Install dependencies

```bash
  cd server
  npm i
```

Start the server

```bash
  npm run dev
```

Open a new terminal and install react dependencies.

```bash
  npm i
```

Run react app

```bash
  npm run start
```

## Tech Stack

**Client:** React

**Server:** Node, Express

**Databse:** MongoDb

## More about server

* Express
* Mongoose
* JWT
